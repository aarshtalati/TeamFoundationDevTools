Welcome to Team Foundation Dev Tools

Team Foundation Dev Tools ( aka TFDT ) is a Free ( as in beer ! ) & Open Source utility software distributed under GNU GPL v2. This software is running on .NET platform 4.5.2

Planned Features :

1. Auto Detect

  • detects and lists all pre configured TFS connections and gives the opportunity to select / switch.
  • also let's you manually connect to a connection ( just in the case )

2. Recursive Find

  • every match is displayed along with Last Check-in Date, Commuted By, Change-Set # & optionally check in comment
  • finds for a particular Wild-Card search for file names on ALL projects' ALL files.
  • you may also narrow down the results for a particular sub-string in a path to that file.
  • screen output has color codded output to differentiate between no match, match and path filter results.

Preferences :

  • search results may be directed to screen and / or a txt file
  • you have the choice to choose between summary and detailed output.
  • you have granular control over switching between summarized / detailed output for screen / txt output
